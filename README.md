
# Simple-Python-GUI-Program-Miles-to-Kilometer

It's a Simple Python Script which uses Python's `Tkinter`  Package to Create a simple GUI Application for Miles to Kilometer Converter


## Formula 
The distance in kilometers is calculated by multiplying the distance in miles by 1.609344. The conversion ratio utilized in the formula is 1.609344 kilometers because one mile equals 1.609344 kilometers.

## Requirement
    1. Python 3
    2. Tkinter - Library

## Run Locally

Clone the project

```bash
  git clone https://github.com/amaithi-sam/Simple-Python-GUI-Program-Miles-to-Kilometer
```

Go to the project directory

```bash
  cd Simple-Python-GUI-Program-Miles-to-Kilometer
```

Install dependencies

```bash
  pip install tk
```

Run the Project

```bash
  python main.py
```
